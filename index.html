JValidator

A simple pluguin made in JQuery focused on easy implementation and customization.

How it works?

Just call the jvalidator.js script in your html file and beware of the call order of the scripts, always the jQuery before. So you can easily use it in the html file or in a external js script, calling the two methods formValidation and inputsValidation.

Explaning the methods:

  • formValidation - Will check all the inputs in a selected (by name,id and etc.. you choose) form and focus the first invalid field.
    sintax: $(form).formValidation();
  • inputsValidation - That method will attach the event "on blur" in all inputs inside a selected form, so validation will be executed when the focus get out of field.
    sintax: $(form).inputsValidation();

Tooltips use explanation

To show a feedback to user about what is wrong or missing in the field is showed a tooltip, but you can change private methods inside plugin to work in another way. You need follow some rules to it work, after the input put a span with the class "tooltip" and just it.

sintax:
<input type="text" name="txt" value="" class="form-element" required>
<span class="tooltip"></span>
